# data

## Concepts
- [agent-result](agent-result.md) -- The JSON output every agent returns — a score, sub-scores, a list of findings with severity/effort/impact, and optional artifacts.
- [audit-report](audit-report.md) -- The final multi-page audit output — headline score, per-page deep results, site-wide coverage stats, before/after visibility, and an LLM-written executive summary.
- [site-facts](site-facts.md) -- The deterministic, LLM-free snapshot of a single page — every signal the four agents read, produced once by the extraction pipeline.
